## 🚀 项目简介

这是一个 **Surge 规则 / 模块 / 脚本 / 配置集合仓库**，用于：

- 📦 分流规则（Ruleset）
- 🔧 Surge 模块（Module）
- 🧠 脚本增强（JavaScript）
- ⚙️ 一键配置（Config）

---

## 📁 结构说明

- `Rules/` → 分流规则列表
- `Module/` → Surge 模块
- `JavaScript/` → 脚本
- `conf/` → 配置文件

---

## ⚙️ 使用方式

### 1️⃣ 直接导入配置（推荐）
https://raw.githubusercontent.com/Valiumlove/surge-rules/main/conf/PersonalConfiguration.conf

## 🧠 使用说明

- 修改 `[Proxy]` 与 `[Proxy Group]` 后使用
- 支持 iOS / macOS Surge
- 建议按模块化方式使用
